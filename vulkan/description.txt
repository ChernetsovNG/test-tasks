Описание задания (письмо от Прохорова И.И. от 16.05.2018)

Для выполнения потребуется:

JDK 8 (например, jdk1.8.0_152)
IntelliJ IDEA
VisualVM Launcher плагин для IntelliJ IDEA

Инструмент "Java Virtual Machine Monitoring"
в моей конфигурации, например, находится здесь
C:/Program Files/Java/jdk1.8.0_152/bin/jvisualvm.exe

Больше самостоятельно узнать про jvisualvm, думаю, проблем нет )

Задание:

1. Реализовать интерфейс Runnable

    Runnable task = () -> { ... }
    
    код должен загружать CPU арифметическими и т.п. вычислениями на 10-20 секунд
    и занимать под данные какой-то значительный объем памяти (10-100 Mb) 
    например, можно сгенерировать случайный LongStream случайного размера,
    потом к каждому элементу применить некоторую математическую функцию, потом посчитать сумму и т.п.

2. Используя 5 (пять) CompletableFuture, выполнить 20 (двадцать) задач task.

    Желательно одновременно, насколько это возможно в контексте задачи )

3. Дождаться выполнения задач, вывести результат.

Оформить как простое консольное приложение, например

public class Test {

    public static void main(String[] args) {
        ...
        System.out.println(...)
    }
}
